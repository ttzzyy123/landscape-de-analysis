    def f(self, x):
        """
        Evaluates the Scaled Arc-Bowl function for a given input vector x.

        The function is defined as a sum of individual dimensional components:
        f(x) = sum_{i=1}^{dim} (x_i^2 * (1 + C * arctan(k * x_i^2)))

        This creates a bowl-shaped landscape. The x_i^2 term ensures a quadratic base,
        while the (1 + C * arctan(k * x_i^2)) term provides a smooth, non-linear
        scaling factor. As |x_i| increases, arctan(k * x_i^2) smoothly increases
        from 0 towards pi/2, making the function increasingly steeper further from
        the origin.

        The global minimum is at x = [0, ..., 0], where f(x) = 0.

        Args:
            x (np.ndarray): A real-valued sample vector of dimension `self.dim`.

        Returns:
            float: The scalar evaluation of the function at x.
        """
